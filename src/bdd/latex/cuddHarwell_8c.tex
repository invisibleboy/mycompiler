Synopsis [Reads in a matrix in the format of the Harwell-Boeing benchmark suite.]

Description [Reads in a matrix in the format of the Harwell-Boeing benchmark suite. The variables are ordered as follows: $<$blockquote$>$ x$\backslash$[0$\backslash$] y$\backslash$[0$\backslash$] x$\backslash$[1$\backslash$] y$\backslash$[1$\backslash$] ... $<$/blockquote$>$ 0 is the most significant bit. On input, nx and ny hold the numbers of row and column variables already in existence. On output, they hold the numbers of row and column variables actually used by the matrix. m and n are set to the numbers of rows and columns of the matrix. Their values on input are immaterial. Returns 1 on success; 0 otherwise. The ADD for the sparse matrix is returned in E, and its reference count is $>$ 0.]
