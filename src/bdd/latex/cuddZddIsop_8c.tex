Synopsis [Computes an ISOP in ZDD form from BDDs.]

Description [Computes an irredundant sum of products (ISOP) in ZDD form from BDDs. The two BDDs L and U represent the lower bound and the upper bound, respectively, of the function. The ISOP uses two ZDD variables for each BDD variable: One for the positive literal, and one for the negative literal. These two variables should be adjacent in the ZDD order. The two ZDD variables corresponding to BDD variable {\tt i} should have indices {\tt 2i} and {\tt 2i+1}. The result of this procedure depends on the variable order. If successful, Cudd\_\-zdd\-Isop returns the BDD for the function chosen from the interval. The ZDD representing the irredundant cover is returned as a side effect in zdd\_\-I. In case of failure, NULL is returned.]

Side\-Effects [zdd\_\-I holds the pointer to the ZDD for the ISOP on successful return.]
